# vue-storybook
IT_KINGDOMメンバープロジェクト. vue.jsでstorybookを作成する

## ローカルでの起動手順
以下をclone  
```https://github.com/tatsuHigh/vue-storybook.git```  

cloneしたフォルダで以下のコマンドを実行  
```npm install```  

続けて以下のコマンドを実行  
```npm run storybook```  

ブラウザで以下のURLにアクセス  
```http://localhost:6006/```  

## 開発手順
作るものはProjectsにIssueを立ててください。  
最初はいくつか議題を出しておきます。やりたいものにアサインしてください。
